# Air Quality Prediction Platform

End-to-end Machine Learning system that predicts real-time Air Quality Index (AQI)
using Government of India live AQI API and a trained regression model.

## Features
- Historical AQI model training
- Live Government AQI API integration
- Flask backend for real-time prediction
- Web dashboard frontend

## How to Run

### Backend
cd app/backend  
python app.py

### Frontend
Open app/frontend/index.html in browser

## Tech Stack
Python, Pandas, Scikit-learn, Flask, Government AQI API
